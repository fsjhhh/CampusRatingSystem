# 校园评分系统

![校园评分系统Logo](/src/assets/logo.png)

## 项目简介
校园评分系统是一个基于Vue.js 3开发的现代化Web应用，旨在为高校师生提供一个全面、便捷的校园服务评价平台。用户可以对课程、食堂、校内商店等多个方面进行评分和评价，也可以自定义评分类别，满足不同场景的需求。该系统集成了评论、评分、图片上传、数据统计等功能，为校园服务质量的提升提供了数据支持和反馈渠道。

## 主要功能

### 课程评价
- 对教师授课质量、教学内容、课程难度等多维度评分
- 支持课程资料分享和推荐
- 课程评分排行榜和推荐系统

### 食堂评价
- 对食堂菜品质量、价格、服务态度、环境卫生等进行评分
- 支持菜品推荐和点评
- 食堂窗口人气排行榜

### 校内商店评价
- 对校内各商店的商品价格、服务质量、商品种类等进行评分
- 商品推荐和优惠信息分享
- 商店综合评分排名

### 自定义评分类别
- 用户可创建自定义评分类别（如校园设施、宿舍环境等）
- 灵活设置评分维度和评价标准
- 支持社区共享评分模板

### 其他特色功能
- 评论支持图片上传（最多3张），更直观地展示实际情况
- 全面的评分统计和数据可视化展示
- 强大的全文搜索功能，快速找到所需信息
- 用户个人中心，记录评分历史和收藏内容
- 响应式设计，支持移动端和桌面端访问

## 安装与部署

### 环境要求
- Node.js 14.0+
- npm 6.0+

### 安装依赖
```bash
npm install
```

### 开发环境运行
```bash
npm run serve
```
开发服务器将运行在 http://localhost:8080

### 生产环境构建
```bash
npm run build
```

### 代码质量检查
```bash
npm run lint
```

## 模块详解

### 首页 (Home.vue)
首页作为系统入口，展示系统概览和最新评分动态。用户可以查看热门评分对象、最新评论和系统使用统计数据。同时，首页提供快速导航到各个评分模块的入口。

### 课程评价 (Course.vue)
课程评价模块允许学生对已修课程进行多维度评价，包括教学内容、教师教学水平、课程实用性等方面。系统自动统计各课程的评分数据，生成课程排行榜，帮助学生选课和教师改进教学。

### 食堂评价 (Food.vue)
食堂评价模块专注于校内各食堂的评价，用户可以对食品质量、价格、服务态度、环境卫生等方面进行打分和评价。系统支持按食堂和窗口进行分类展示，方便用户查找和比较。

### 商店评价 (Stores.vue)
商店评价模块收集用户对校内商店的评价，包括商品价格、服务质量、商品种类等方面。用户可以分享购物体验和推荐商品，帮助校内商店改进服务和商品供应。

### 自定义评分 (Custom.vue)
自定义评分模块是系统的特色功能，允许用户创建自定义的评分类别，如校园设施、宿舍环境、图书馆服务等。用户可以设定评分维度和评价标准，满足多样化的评价需求。

### 添加评分对象 (AddPost.vue)
添加评分对象页面提供简洁的表单界面，用户可以创建新的评分对象，设置类别、名称、描述、图片等信息。系统支持预览功能，确保信息准确后提交。

### 详情页 (Detail.vue)
详情页展示特定评分对象的详细信息，包括基本信息、评分统计、用户评论等。用户可以在此页面添加自己的评分和评论，上传图片佐证，参与讨论交流。

## 使用指南

### 基本操作流程
1. **浏览评分**: 首页展示系统概览和热门评分，通过顶部导航可进入不同的评分类别
2. **查看详情**: 点击任意评分对象可进入详情页，查看详细评价和评分统计
3. **添加评分**: 在详情页可添加自己的评分和评论，支持上传最多3张图片
4. **创建评分对象**: 点击"添加新评分对象"按钮可创建新的评分对象
5. **自定义评分类别**: 在自定义评分页面可创建和管理自定义评分类别

### 搜索功能
系统提供强大的全文搜索功能，用户可以通过关键词搜索评分对象、评论内容或用户名，快速找到所需信息。搜索结果按相关度排序，支持多种筛选条件。

### 数据可视化
系统提供丰富的数据可视化图表，直观展示评分统计、趋势变化和对比分析。用户可以通过这些图表快速了解校园服务的整体情况和评价趋势。

## 数据存储
系统目前使用浏览器的LocalStorage进行数据存储，适合小规模使用和原型演示。在实际部署时，建议接入后端数据库（如MySQL、MongoDB等）以支持更大规模的数据存储和更复杂的查询需求。
